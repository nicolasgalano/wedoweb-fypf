## Local setup

1. **Go into the root of the project** with the next line:
	```shell
	$ cd html
	```

2. **Install JS dependencies**
	```shell
	# @ root-folder/html
	$ npm install
	```

3. **Run Gulp serve**
	```shell
	# @ root-folder/html
	$ gulp watch
	```
